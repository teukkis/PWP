# Shopping List

Includes six functions, handleDelete(), handleEdit(), displayEdit(), addToPantry(), renderRows() and renderHeaders().

### handleEdit()

Async function

#### description

Calls function sendData(), which sends a PUT request to the server.

#### input parameters

Takes "event" parameter.

### handleDelete()

Async function

#### description

Calls function getResource(), which sends a GET request to the server.

#### input parameters

Takes "event" and shopping list item's "name" as parameters.

### displayEdit()

#### description

Sets edit variable true, which causes re-render and new textfield for food item editing are displayed.


### renderRows()

#### description

Renders a list of shoppinglist's food items and sets all the information to these items for later use.

### renderHeaders()

#### description

Renders all column headers for the shopping list item's table.


### addToPantry()

Async function

#### description

Calls function sendData(), which sends a POST request to the server.

#### input parameters

Takes "event" and shopping list item as parameters.
