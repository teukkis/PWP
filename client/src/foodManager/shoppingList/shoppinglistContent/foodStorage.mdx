# Shopping List

Includes six functions, handleAdd(), renderRows() and renderHeaders().


### renderRows()

#### description

Renders a list of food items in the database and sets all the information to these items for later use.

### renderHeaders()

#### description

Renders all column headers for the food storage table.


### handleAdd()

Async function

#### description

Calls function sendData(), which sends a POST request to the server.
Food item will be added to the shopping list.

#### input parameters

Takes "event", item "name" and item "id" as parameters.
