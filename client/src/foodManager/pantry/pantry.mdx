# Pantry

Includes a function for deleting a pantry food item and button to launch the event. Displays all pantry food items.


### handleDelete()

Async function

#### description
Calls function getResource, which sends a DELETE request to the server.

#### input parameters

Takes an "event" parameter and uses redux store to get necessary controls information for the request.
